# Controle de Vendas Android

Aplicativo Android desenvolvido como projeto acadêmico para controle de vendas e estoque.

## Funcionalidades
- Cadastro de produtos
- Listagem e exclusão de produtos
- Relatórios de vendas
- Armazenamento local (Room ou SQLite)

## Tecnologias
- Android Studio
- Kotlin
- RecyclerView
- Material Design

## Como executar
1. Clonar o repositório
2. Abrir o projeto no Android Studio
3. Executar o aplicativo em um emulador ou dispositivo real
